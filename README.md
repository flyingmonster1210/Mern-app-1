# A Mern App

## Set up the .env file

NODE_ENV = development
PORT = 5000
MONGO_URI = YOUR_MONGO_URI
JWT_SECRET = ANY_STRING

## Run the app

1. yarn run dev
   this command run both the backend and frontedn app at the same time
2. yarn run server
   run only the backend app
3. yarn run client
   run only the frontend app
